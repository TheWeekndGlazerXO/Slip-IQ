FAQ

Common Questions

Are the odds and matches actually real? +
Yes. Odds are pulled live from The Odds API which aggregates Bet365, Pinnacle, Betway and other major bookmakers. Fixtures come from SportMonks. The win probabilities are generated by our own Monte Carlo model — not copied from bookmakers.
How does the AI analysis work? +
We send each match's ELO ratings, form, xG values, lineup data, and odds to OpenAI's GPT-4o model with a structured prompt. It returns a tactical breakdown, recommendation, key factors, and upset analysis in real time.
What does the auto parlay builder actually do? +
You set a target odds multiplier (e.g. 5x). Oracle scans all upcoming matches, ranks them by confidence score and value detection, then builds the combination of selections that hits your target with the highest probability of winning.
Is this a guarantee I'll win? +
No. Slip IQ provides data-driven analysis and probability modelling — it does not guarantee outcomes. Betting involves risk and you should only bet what you can afford to lose. Oracle gives you a statistical edge, not certainty.
